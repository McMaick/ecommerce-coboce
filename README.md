# Sistema E-commerce Cerámica COBOCE

Sistema de comercio electrónico desarrollado para la distribuidora **Cerámica COBOCE**, ubicada en Cobija, Bolivia.

## Stack tecnológico

- PHP 8
- MySQL
- HTML5 / CSS3 / JavaScript
- Bootstrap 5

## Módulos implementados

- Catálogo de productos
- Carrito de compras
- Checkout en 4 pasos: Entrega, Puntos, Pago, Confirmación
- Control de inventario
- Sistema de delivery por zonas configurables
- Sistema de puntos / fidelidad
- Login, registro y recuperación de contraseña
- Panel de administración completo (productos, categorías, pedidos, usuarios, inventario, reportes, configuración de tienda)

## Datos de la tienda física

- **Dirección:** Av. Pando, a lado del Centro de Salud Santa Clara
- **Ciudad:** Cobija, Bolivia
- **WhatsApp:** 73943006
- **Horario:** Lunes a sábado 8:00–18:00 | Domingo cerrado

## Instalación

1. Importar el script de `base de datos/coboce.sql` en MySQL.
2. Configurar las credenciales de conexión en `config/config.php`.
3. Servir el proyecto con un servidor compatible con PHP 8 (Apache/XAMPP).
4. Acceder a `index.php` desde el navegador.
